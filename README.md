# Big Data in CRM

## Overview

This project explores the application of Big Data techniques in Customer Relationship Management (CRM) to analyze customer data, identify trends, and support data-driven business decisions.

## Objectives

* Analyze customer data
* Discover customer behavior patterns
* Improve customer segmentation
* Support business decision-making through data analysis

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Google Colab

## Project Workflow

* Data Collection
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Customer Insights
* Business Recommendations

## Repository Contents

* `Big_Data_in_CRM.ipynb` – Complete Google Colab notebook containing the CRM analysis.
* `requirements.txt` – Required Python libraries.
